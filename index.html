Bounty of Kings

Developed by Justin Cordero

What is "Bounty of Kings"?

Bounty of kings is a text advendture game that allows the player to make certain choices that may effect certain aspects of the game. The first choice, being most important, is picking the name.

About the Software

The game is devloped in python and is meant to be played on any software that can be played on any device that runs python. A way to host it to the public has not been decided yet.

Beta Features

The key features that are being focused on for the beta is a converstion with a questgiver and the first part of the dungeon available to players. This will include dialogue options, rooms that have many possibilities, and combat.

What to Expect

The hopes for the full game are to have multiple endings, easter eggs to my favorite games, an inventory system, and more quests.

All Tech Features

The basic feature will include loops and lists to pick things based on certain choices. The more advanced things are a random number generator and a storage for input responses to be carried throughout the game.
